# 걸음수 관리 웹페이지

이 웹페이지는 2026년 3월 11일부터 2027년 3월 10일까지 4,000,000보를 목표로 하는 걸음수 관리 애플리케이션입니다.

## 주요 기능

- 기준 날짜 선택
  - 기본값은 어제 날짜
  - 날짜를 누르면 달력 팝업이 열리고 기준 날짜 변경 가능
  - 선택 가능한 범위: 2026-03-11 ~ 2027-03-10
- 누적 걸음수 저장
  - 입력한 값은 선택한 기준 날짜의 누적 걸음수로 저장
  - 데이터는 날짜별 히스토리 형태로 브라우저 로컬 스토리지에 저장
- 통계 자동 계산
  - 기준일
  - 걸어온 날
  - 걸어온 걸음
  - 평균 하루 걸음
  - 남은 날
  - 남은 걸음수
  - 평균 걸어야 될 걸음수
- 누적통계 팝업 그래프
  - 토요일 기준 최근 10개 누적 걸음수 꺾은선
  - 1년 400만보 목표 달성을 위한 예상 누적 걸음수(기준선) 함께 표시
- 최근통계 팝업 그래프
  - 최근 7일 일일 걸음수 표시
  - 계산 방식: 해당일 누적 - 직전일 누적
  - 직전일 데이터가 없으면 이전/다음 누적 데이터 간 차이를 날짜 수로 나눠 추정

## 사용 방법

1. 웹페이지를 엽니다.
2. 기준 날짜를 클릭해 필요한 날짜로 선택합니다.
3. '어제까지 누적된 걸음수'에 값을 입력합니다.
4. '확인' 버튼을 눌러 선택한 기준 날짜로 저장합니다.
5. 메인 통계 카드에서 현재 진행 상황을 확인합니다.
6. '누적통계' 버튼으로 최근 10주 누적 추이를 확인합니다.
7. '최근통계' 버튼으로 최근 7일 일일 걸음수를 확인합니다.

## 데이터 저장 방식

- 저장 키: `walkingHistory`
- 형식: `{ "YYYY-MM-DD": 누적걸음수 }`
